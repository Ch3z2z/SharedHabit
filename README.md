# SharedHabits

SharedHabits - это веб-приложение для совместного отслеживания привычек. Пользователи могут создавать привычки, делиться ими с друзьями, приглашать участников и отслеживать прогресс выполнения.

## Особенности

- **Создание и управление привычками**: Пользователи могут создавать личные привычки с описанием и настройками.
- **Совместное использование**: Возможность делиться привычками с другими пользователями.
- **Приглашения**: Система приглашений для добавления участников в общие привычки.
- **Отслеживание прогресса**: Логирование выполнения привычек по датам.
- **Веб-интерфейс**: Современный iOS-подобный интерфейс с адаптивным дизайном.
- **Аутентификация**: JWT-based аутентификация для безопасного доступа.

## Технологии

- **Backend**: Flask (Python)
- **База данных**: PostgreSQL с SQLAlchemy ORM
- **Аутентификация**: Flask-JWT-Extended
- **Frontend**: HTML, CSS, JavaScript (с Bootstrap-подобными стилями)
- **Шаблоны**: Jinja2

## Установка

### Предварительные требования

- Python 3.8+
- PostgreSQL
- Git

### Шаги установки

1. **Клонируйте репозиторий:**
   ```bash
   git clone <repository-url>
   cd SharedHabbits
   ```

2. **Создайте виртуальное окружение:**
   ```bash
   python -m venv venv
   ```

3. **Активируйте виртуальное окружение:**
   - Windows:
     ```bash
     venv\Scripts\activate
     ```
   - Linux/Mac:
     ```bash
     source venv/bin/activate
     ```

4. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

5. **Настройте базу данных:**
   - Создайте базу данных PostgreSQL с именем `SharedHabit`
   - Обновите настройки подключения в `app/config.py` при необходимости

6. **Запустите приложение:**
   ```bash
   python run.py
   ```

Приложение будет доступно по адресу `http://localhost:5000`

## Конфигурация

Основные настройки находятся в файле `app/config.py`:

- `SQLALCHEMY_DATABASE_URI`: URL подключения к PostgreSQL
- `JWT_SECRET_KEY`: Секретный ключ для JWT токенов
- `SQLALCHEMY_TRACK_MODIFICATIONS`: Отключение отслеживания изменений SQLAlchemy

## API Endpoints

### Аутентификация
- `POST /api/login` - Вход в систему
- `POST /api/register` - Регистрация нового пользователя

### Привычки
- `GET /api/habits` - Получить список привычек пользователя
- `POST /api/habits` - Создать новую привычку
- `GET /api/habits/<id>` - Получить детали привычки
- `PUT /api/habits/<id>` - Обновить привычку
- `DELETE /api/habits/<id>` - Удалить привычку

### Логи привычек
- `GET /api/habits/<id>/logs` - Получить логи выполнения
- `POST /api/habits/<id>/logs` - Добавить запись о выполнении

### Приглашения
- `POST /api/habits/<id>/invite` - Отправить приглашение
- `GET /api/invites` - Получить список приглашений
- `POST /api/invites/<id>/accept` - Принять приглашение

## Использование

1. **Регистрация**: Создайте аккаунт с уникальным именем пользователя и email
2. **Вход**: Войдите в систему с вашими учетными данными
3. **Создание привычки**: Нажмите "Создать привычку" и заполните детали
4. **Совместное использование**: Сделайте привычку общей и пригласите друзей
5. **Отслеживание**: Отмечайте выполнение привычек ежедневно

## Разработка

Для разработки рекомендуется использовать режим отладки Flask:

```bash
export FLASK_ENV=development
python run.py
```

## Troubleshooting

### Проблемы с базой данных
- Убедитесь, что PostgreSQL запущен
- Проверьте правильность URL подключения в `config.py`
- Создайте базу данных `SharedHabit` заранее

### Ошибки зависимостей
- Убедитесь, что виртуальное окружение активировано
- Переустановите зависимости: `pip install -r requirements.txt`

### Проблемы с JWT
- Проверьте, что `JWT_SECRET_KEY` установлен в `config.py`
- Очистите cookies браузера при проблемах с аутентификацией
